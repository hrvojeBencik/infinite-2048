## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, chip internal padding |
| sm | 8px | Tile grid cell gap, inline element spacing |
| md | 16px | Default padding (cards, list tiles, dialog body) |
| lg | 24px | Dialog padding horizontal, button horizontal padding |
| xl | 32px | Screen-level section gaps |
| 2xl | 48px | Major layout breaks |
| 3xl | 64px | Page-level vertical spacing |

Exceptions:
- Button vertical padding: 14px (established in ElevatedButtonTheme ‚Äî keep as-is, source: `app_theme.dart` line 45)
- Touch targets for swipe gesture: 44px minimum height (GestureDetector zones, iOS HIG)
- Swipe block timer: 420ms (400ms merge animation + 20ms buffer ‚Äî not a spacing value, documented here for completeness)

Source: `lib/core/theme/app_theme.dart` (button padding), CONTEXT.md D-01 (animation timing)

## Color

| Role | Token | Hex | Usage |
|------|-------|-----|-------|
| Dominant (60%) | AppColors.background | `#0A0E21` | Scaffold background, native splash background, grid background |
| Secondary (30%) | AppColors.surface | `#16213E` | Cards, dialogs, game over dialog, level complete dialog background |
| Surface variant | AppColors.surfaceLight | `#1F2F50` | XP bar track background, snackbars, secondary cards |
| Accent (10%) | AppColors.primary | `#6C63FF` | XP bar fill gradient start, confetti particle color 1, merge animation glow, primary CTA buttons |
| Accent secondary | AppColors.secondary | `#FFD700` | Confetti particle color 2, premium badge, star elements in level complete |
| Success semantic | AppColors.success | `#00E676` | Confetti particle color 3, level complete check mark |
| Destructive | AppColors.error | `#FF5252` | Destructive actions only (none introduced in this phase) |

Accent (AppColors.primary) is reserved for:
- XP bar animated fill (gradient: `AppColors.primaryLight` ‚Üí `AppColors.primary`)
- Confetti particle burst (one of 5 colors)
- Tile merge glow effect (existing ‚Äî no change)
- Primary CTA buttons (existing ‚Äî no change)

Confetti palette (5 colors, all from AppColors or complementary):
1. `AppColors.secondary` ‚Äî `#FFD700` gold
2. `AppColors.primary` ‚Äî `#6C63FF` purple
3. `AppColors.success` ‚Äî `#00E676` green
4. `Color(0xFFFF6B6B)` ‚Äî coral (complements brand, warm contrast)
5. `Color(0xFF48DBFB)` ‚Äî cyan (cool contrast, distinct from other particles)

Source: RESEARCH.md Pattern 3 (confetti colors), `lib/core/theme/app_colors.dart`, CONTEXT.md D-11 (splash background)

## Animation Contract

This section is specific to Phase 3 and specifies every animation's visual parameters. The executor must not deviate from locked values (marked L). Values marked D are at Claude's discretion within constraints.

### ANIM-01: Tile Merge Scale-Pop

| Parameter | Value | Source |
|-----------|-------|--------|
| Curve | easeOutCubic | CONTEXT.md D-01 (L) |
| Duration | 400ms | CONTEXT.md D-01 (L) |
| Keyframes | 0.85 ‚Üí 1.18 ‚Üí 0.95 ‚Üí 1.0 | CONTEXT.md D-01 (L) |
| Swipe block duration | 420ms (400ms + 20ms buffer) | RESEARCH.md Pattern 1 (D) |
| Block implementation | `_boardAnimating` local flag in `_GamePageState` | RESEARCH.md Pattern 1 (D) |

No visual changes to the animation itself. The fix is behavioral: swipes are silently ignored while `_boardAnimating == true`.

### ANIM-02: Screen Transitions

| Transition type | Pattern | Duration | Curve | Source |
|-----------------|---------|----------|-------|--------|
| Lateral (route-to-route) | Fade + slight slide-right via `_buildTransitionPage()` | 300ms | existing | CONTEXT.md D-03 (L ‚Äî no change) |
| Full-screen modal (level complete, game over) | Slide-up from bottom via `showGeneralDialog` + `SlideTransition` | 350ms | easeOutCubic | CONTEXT.md D-04 (L) |

Slide-up offset: `Offset(0, 1)` ‚Üí `Offset.zero`.

`showGeneralDialog` must include `routeSettings: const RouteSettings(name: '/level-complete')` and `routeSettings: const RouteSettings(name: '/game-over')` respectively, to preserve analytics observer behavior.

### ANIM-03: Haptic Feedback on Merge

| Parameter | Value | Source |
|-----------|-------|--------|
| Trigger | `state.lastMergeCount > 0` in `_triggerJuiceEffects()` | CONTEXT.md D-05 (L) |
| Service call | `sl<HapticService>().merge()` | CONTEXT.md D-05 (L) |
| Timing | Fires at same moment as merge juice effects | RESEARCH.md (D) |

Haptic is inaudible/invisible ‚Äî no visual component. Verify on physical device that it fires for single-tile merges (not only combos).

## Copywriting Contract

Phase 3 is a polish phase with no new screens. There are no new copy-heavy surfaces. The copywriting contract covers states introduced or modified by animation work.

| Element | Copy |
|---------|------|
| Primary CTA (level complete dialog) | "Continue" (existing ‚Äî no change) |
| Primary CTA (game over dialog) | "Try Again" (existing ‚Äî no change) |
| Native splash state | No text ‚Äî icon only, centered on dark background |
| Swipe-blocked state | Silent ‚Äî no copy, no toast. Input is silently ignored during animation |
| Haptic feedback state | No visual copy ‚Äî physical only |
| Level complete celebration heading | "Level Complete!" (existing ‚Äî no change) |
| Game over heading | "Game Over" (existing ‚Äî no change) |
| Empty state | Not applicable ‚Äî this phase modifies no data-driven lists |
| Error state | Not applicable ‚Äî animation failures degrade silently (animation simply doesn't play) |
| Destructive actions | None introduced in this phase |
