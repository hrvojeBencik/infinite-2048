  <action>
Convert XpBar from StatelessWidget to StatefulWidget with an explicit AnimationController per D-08 and Research Pitfall 5:

1. Change `class XpBar extends StatelessWidget` to `class XpBar extends StatefulWidget`. Add `State<XpBar> createState() => _XpBarState();`.

2. In `_XpBarState`, add `SingleTickerProviderStateMixin` and declare:
   - `late AnimationController _controller;` — duration 600ms, curve easeOutCubic
   - `double _previousProgress = 0.0;` — stores previous fill value
   - `late Animation<double> _animation;` — drives the fill width

3. In `initState()`:
   - Create `_controller = AnimationController(vsync: this, duration: const Duration(milliseconds: 600));`
   - Compute initial progress: `final progress = widget.profile.xpRequiredForNextLevel > 0 ? widget.profile.xpForCurrentLevel / widget.profile.xpRequiredForNextLevel : 1.0;`
   - Set `_previousProgress = progress.clamp(0.0, 1.0);`
   - Create `_animation = Tween<double>(begin: 0.0, end: _previousProgress).animate(CurvedAnimation(parent: _controller, curve: Curves.easeOutCubic));`
   - Call `_controller.forward();` to animate from 0 to current on first render.

4. In `didUpdateWidget(XpBar oldWidget)`:
   - Compute new progress from `widget.profile`
   - If new progress != `_previousProgress`:
     - Create new `_animation = Tween<double>(begin: _previousProgress, end: newProgress.clamp(0.0, 1.0)).animate(CurvedAnimation(parent: _controller, curve: Curves.easeOutCubic));`
     - `_controller.forward(from: 0.0);`
     - `_previousProgress = newProgress.clamp(0.0, 1.0);`

5. In `dispose()`: `_controller.dispose();`

6. In `build()`: Wrap the LayoutBuilder's fill bar with `AnimatedBuilder(animation: _animation, ...)`. Replace `progress.clamp(0.0, 1.0)` with `_animation.value` for the `progressWidth` calculation:
   ```dart
   final progressWidth = constraints.maxWidth * _animation.value;
   ```

7. Keep all other widget structure unchanged (Container wrapper, Row with level text, bar, XP text). The XP text should still show the actual (non-animated) values: `${widget.profile.xpForCurrentLevel} / ${widget.profile.xpRequiredForNextLevel}`.

Per UI-SPEC: fill gradient is AppColors.primaryLight to AppColors.primary (already correct). Track color is AppColors.background.withAlpha(150) (already correct). Bar height is 10px (already correct).
